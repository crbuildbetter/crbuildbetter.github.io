AI / Data Science Engineer

เปิดรับสมัคร AI Data Science

buildbetter กำลังเปิดรับสมัคร AI / Data Science Engineer ที่มีความสามารถสูงและพร้อมลุยงานรอบด้านมาร่วมทีม — ดูแลวงจรชีวิตของระบบที่ขับเคลื่อนด้วยข้อมูลตั้งแต่ต้นจนจบ ตั้งแต่ Pipeline ข้อมูลดิบไปจนถึงการนำโมเดลอัจฉริยะขึ้นสู่ Production

ดูรายละเอียดตำแหน่งงาน

เกี่ยวกับเรา

บริษัท buildbetter เป็นบริษัทคู่ค้าของ ProGaming ซึ่งเป็นผู้เชี่ยวชาญด้านการพัฒนาซอฟต์แวร์ AI application และ AI solution ในรูปแบบต่าง ๆ เรากำลังมองหา AI / Data Science Engineer ที่มีความสามารถสูงและพร้อมลุยงานรอบด้านมาร่วมทีม

ในบทบาทนี้ คุณจะได้เป็นผู้ดูแลวงจรชีวิตผลิตภัณฑ์ (Lifecycle) ของระบบที่ขับเคลื่อนด้วยข้อมูลตั้งแต่ต้นจนจบ คุณไม่ได้ทำแค่สร้างโมเดลในเชิงทฤษฎีเท่านั้น แต่จะได้สร้าง Pipeline ข้อมูลดิบและผลักดันโมเดลอัจฉริยะเข้าสู่สภาพแวดล้อมการใช้งานจริง (Production) หากคุณชอบความท้าทายในการออกแบบสถาปัตยกรรมที่ซับซ้อนและปรับใช้แอปพลิเคชัน AI ในโลกจริง เราอยากร่วมงานกับคุณ!

ความรับผิดชอบ

  • Data Collection (การจัดเก็บข้อมูล): ออกแบบ สร้าง และดูแล Pipeline การเก็บรวบรวมข้อมูลแบบอัตโนมัติที่รองรับการขยายตัว (Scalable) จากแหล่งข้อมูลต่าง ๆ เช่น ฐานข้อมูล, API, System Logs และการเชื่อมต่อกับระบบภายนอก (Third-party Integrations)
  • Data Cleansing & Preparation (การทำความสะอาดและเตรียมข้อมูล): ออกแบบกระบวนการทำความสะอาดข้อมูล (Preprocessing) การแปลงข้อมูล (Parsing) และการทำให้ข้อมูลเป็นมาตรฐาน (Normalization) รวมถึงทำ Feature Engineering และจัดการกับข้อมูลที่ขาดหายหรือไม่สอดคล้องกัน เพื่อสร้างชุดข้อมูลที่มีคุณภาพสูง
  • Modeling (การสร้างโมเดล): คัดเลือก ออกแบบ และฝึกฝน (Train) โมเดล Machine Learning, Deep Learning หรือโมเดลทางสถิติ ที่เหมาะสมที่สุดในการแก้โจทย์ทางธุรกิจและฟีเจอร์ของแอปพลิเคชัน
  • Evaluation (การประเมินผล): สร้างกรอบการประเมินผลที่รัดกุม ตรวจสอบโมเดลกับเกณฑ์มาตรฐาน (Benchmarks) ทั้งในด้านประสิทธิภาพ (Performance) ความแม่นยำ (Accuracy) ความสามารถในการรองรับการขยายตัว (Scalability) และความลำเอียงของข้อมูล (Bias) ก่อนนำไปใช้งานจริง
  • Deployment (การติดตั้งและเปิดใช้งาน): ทำ Containerization ปรับใช้ (Deploy) และขยายขนาดโมเดล AI สู่สภาพแวดล้อมการใช้งานจริง (Production) ผ่าน Microservices, REST/GraphQL APIs และเครื่องมือ MLOps เพื่อให้มั่นใจว่าสามารถผสานรวมกับแอปพลิเคชันและเกมของเราได้อย่างราบรื่น
  • Monitoring & Optimization (การติดตามและเพิ่มประสิทธิภาพ): ติดตามตรวจสอบ Model Drift และประสิทธิภาพในระบบจริงอย่างต่อเนื่อง พร้อมทั้งตั้งค่าลูปการฝึกฝนโมเดลซ้ำแบบอัตโนมัติ (Automated Retraining Loops) และรอบการปรับปรุงประสิทธิภาพ
  • Collaboration (การทำงานร่วมกัน): ร่วมมือกับนักพัฒนาซอฟต์แวร์ (Software Developers) ผู้จัดการผลิตภัณฑ์ (Product Managers) และผู้มีส่วนเกี่ยวข้อง เพื่อเปลี่ยนเป้าหมายทางธุรกิจให้เป็นโซลูชันทางเทคนิคที่สามารถใช้งานได้จริง

คุณสมบัติ

  • วุฒิการศึกษาระดับปริญญาตรีหรือปริญญาโท สาขาวิทยาการคอมพิวเตอร์ (Computer Science), วิทยาศาสตร์ข้อมูล (Data Science), สถิติ, คณิตศาสตร์ หรือสาขาเชิงปริมาณที่เกี่ยวข้อง
  • มีประสบการณ์ตรง (Hands-on Experience) ครอบคลุมวงจรชีวิตของการจัดการ Pipeline ข้อมูลและการพัฒนา Machine Learning ทั้งระบบ
  • มีความเชี่ยวชาญในภาษา Python และเครื่องมือในระบบนิเวศที่เกี่ยวข้อง (Pandas, NumPy, Scikit-Learn, SciPy)
  • เข้าใจอย่างลึกซึ้งในเฟรมเวิร์กหลักของ ML/DL (เช่น TensorFlow, PyTorch, Keras)
  • มีความสามารถที่แข็งแกร่งในด้าน SQL และการออกแบบฐานข้อมูล (สถาปัตยกรรม Relational และ NoSQL)
  • มีประสบการณ์เกี่ยวกับ Containerization และสถาปัตยกรรมคลาวด์ (Docker, Kubernetes, AWS, GCP หรือ Azure) สำหรับการให้บริการโมเดลในวงกว้าง
  • หากมีความคุ้นเคยกับแนวคิด MLOps และ CI/CD Pipelines สำหรับ Machine Learning จะพิจารณาเป็นพิเศษ
  • มีแนวคิดในการแก้ปัญหาที่ดี (Problem-solving Mindset) มีทักษะการวิเคราะห์ และมีทัศนคติเชิงรุกในการทำงานร่วมกันเป็นทีม
